# StellarNotes

A simple Soroban smart contract that saves and retrieves notes on the Stellar blockchain.

---

## Problem

Students may lose important notes or files when changing devices or using unstable internet connections.

---

## Solution

StellarNotes allows users to save and retrieve short notes on-chain using Soroban smart contracts on Stellar.

---

## Timeline

Built during a Stellar / Soroban bootcamp activity.

---

## Stellar Features Used

- Soroban Smart Contracts
- Stellar Blockchain Storage

---

## Vision and Purpose

The project demonstrates how Soroban smart contracts can store and retrieve simple user data on-chain using Stellar’s fast and low-cost network.

---

## Prerequisites

- Rust
- Soroban CLI
- Freighter Wallet
